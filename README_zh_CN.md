# Siyuan Code Tabs

[English](./README.md)

## 简介

这个思源插件允许你将多种语言的代码放在一组标签页下，为你的代码展示提供更优雅的解决方案。通过简洁的语法，你可以轻松创建带有标签页的代码块，支持多种编程语言。

## 已知问题

- 不保证适配所有第三方主题
- 导出为markdown或html等格式时所有标签页样式会丢失，只有导出pdf或图片时标签页才能正常显示，且无法切换标签页
- 更换主题时代码标签页的样式可能会出现异常，这时请尝试以下方法：
  - 切换外观模式
  - 切换主题
  - 关闭已打开的文档再重新打开
  - 重启思源笔记后再次尝试以上方法
- **_目前仅3.5.0及以上版本可正常使用_**

## 提示

- 需要在 设置 -> 编辑器 中打开 `允许执行HTML块内脚本` 后插件才能正常运行

- 标签的所有原始数据都以`base64`编码的形式存储在自定义属性中，标签页意外出错时只要自定义属性没丢失就还能还原
- v2.0.0 提供标签页编辑面板，已移除“切回代码块”入口，请使用右上角“编辑”按钮修改内容

## 使用说明

1. 通过斜杠菜单输入 `标签页`/`tab`/`bq` 插入一个默认标签页，也可以选中多个代码块后使用块菜单中的“块：合并选中代码块”生成标签页。
2. 点击标签页右上角 **编辑** 按钮，修改标签标题、语言与代码内容，也支持新增和删除标签。
   - 语言为 `markdown-render` 时会使用思源内置 Lute 处理 markdown，支持 KaTeX / Mermaid / Graphviz / PlantUML / ABC 扩展。
3. 点击标签页右上角的 **设为默认** 按钮，可将当前展示的标签设为默认展示标签
4. 点击标签页右上角的 **复制** 按钮可一键复制当前标签代码。

5. 块菜单中提供更多操作入口：
   - `块：合并选中代码块`：将选中的多个代码块合并为标签页
   - `块：拆分标签页为代码块`：将标签页拆分为多个代码块
   - `文档：拆分标签页为代码块`：将当前文档中所有标签页拆分为代码块
6. 设置面板中提供全局入口：`将所有代码标签页拆分为标准代码块`（用于在弃用插件时能方便还原）。
7. 设置面板提供旧版标签页迁移入口：`迁移旧标签页`（将旧版本标签页升级为新版本）。
8. 标签过多时一部分标签会被隐藏，可以点击最右侧的更多标签进行查看。
9. 效果演示  
   ![图5](./images/demo.gif)
10. 由于第三方主题过于复杂，0.7.0版本增加了一个自定义配置文件，可以让用户自行适配需要使用的主题，配置文件的路径在`SiYuan工作空间/data/plugins/code-tabs/custom/theme-adaption.yaml`，仿照其中的示例并配合使用开发者工具即可完成对主题的适配。也可以将自己适配好的主题提交 pr 到本仓库，仓库中适配文件的路径为 `/public/asset/theme-adaption.yaml`
